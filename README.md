# Titanic-Data-Cleaning-Visualization
Data Cleaning and Visualization project using the Titanic dataset with Python, Pandas, and Matplotlib.
Titanic Data Cleaning & Visualization

Project Overview

This project focuses on cleaning, preprocessing, and visualizing the Titanic dataset using Python.

Technologies Used

- Python
- Pandas
- Matplotlib
- Google Colab

Data Cleaning Tasks

- Handled missing values
- Checked and removed duplicates
- Processed inconsistent data
- Prepared data for analysis

Visualizations

- Age Distribution
- Gender Distribution
- Survival Count

Key Insights

- Most passengers were between 20–35 years old.
- Male passengers outnumbered female passengers.
- More passengers did not survive than survived.

- Dataset
